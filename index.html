Latest Project

MeowMarket

MeowMarket is an ecommerce website offering a seamless and delightful cat adoption service, helping new kitten owners find the perfect suppliers to set up a loving home. As part of the MeowMarket project, I designed both back-end and front-end functionalities, including order handling, the cart filling, cart management, and facilitating purchases for visitors and registered users, and handling edge cases with asynchronous database queries. I adeptly managed a four-member engineering team, ensuring a smooth development process and successful project deployment. Check out the MeowMarket Website, and the Github Repository.
